Caesar cipher encryption technique created by Julius Caesar in an effort to protect classified documents. It works by shifting each letter of the message "x" amount of places from its position in the alphabet. This program has a take of Caesar cipher using a key that changes the amount each character is shifted per iteration. It is capable of encrypting messages as well as decrypting messages that it has encrypted.
